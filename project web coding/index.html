Tentang Saya

Halo! Nama saya Ragah Mujahidin, dan saya saat ini sedang menempuh pendidikan di Universitas Lampung. Saya adalah seorang pemula yang antusias dalam dunia programming dengan semangat yang tinggi untuk terus belajar dan berkembang. Selain itu, saya juga seorang penulis lepas yang gemar berbagi pengetahuan dan pengalaman melalui tulisan. Meskipun saya baru memulai perjalanan ini, saya telah mengumpulkan beberapa pengalaman berharga dalam berbagai teknologi yang akan saya ceritakan lebih lanjut.

Pengalaman dan Keterampilan

  • HTML dan CSS: Selama satu minggu terakhir, saya telah bekerja intensif dengan HTML dan CSS. Saya belajar bagaimana membuat struktur halaman web yang efisien serta mendesain tampilan yang menarik dan responsif. Proyek-proyek yang saya kerjakan mencakup pembuatan halaman landing page sederhana dan beberapa layout dasar.
  • PHP: Dalam tiga hari yang singkat, saya telah mengenal dasar-dasar PHP, sebuah bahasa pemrograman server-side yang digunakan untuk membuat halaman web yang dinamis. Saya telah mempelajari cara mengintegrasikan PHP dengan HTML untuk membuat form dan menyimpan data ke dalam database.
  • WordPress: Pengalaman saya selama satu minggu dengan WordPress telah membuka mata saya terhadap kemudahan dan fleksibilitas platform ini dalam pembuatan dan manajemen situs web. Saya telah mempelajari cara menginstal dan mengkonfigurasi WordPress, menggunakan tema dan plugin untuk memperkaya fungsi dan tampilan situs, serta membuat beberapa posting dan halaman.

Pendidikan

Saya saat ini sedang menempuh studi di Universitas Lampung, di mana saya mendapatkan dukungan dan bimbingan yang luar biasa dari dosen-dosen yang kompeten. Pendidikan formal yang saya jalani sangat membantu saya memahami konsep-konsep dasar dalam teknologi informasi dan pengembangan web.

Visi dan Misi

Sebagai seorang programmer pemula, saya memiliki visi untuk menjadi seorang Cyber Security Consultant yang handal dan terpercaya. Saya ingin membantu organisasi dan individu melindungi data dan sistem mereka dari ancaman siber yang semakin kompleks dan canggih.

Misi saya adalah:

  1. Terus Belajar dan Mengasah Keterampilan: Saya berkomitmen untuk terus belajar tentang keamanan siber, mengikuti perkembangan terbaru dalam teknologi, dan mendapatkan sertifikasi yang relevan untuk memperkuat pengetahuan dan keterampilan saya.
  2. Praktik dan Pengalaman Nyata:Saya akan mencari peluang untuk terlibat dalam proyek-proyek keamanan siber, baik melalui magang, kerja paruh waktu, atau proyek mandiri, untuk mendapatkan pengalaman praktis yang berharga.
  3. Kolaborasi dan Jaringan Profesional:Saya akan membangun jaringan dengan profesional di bidang keamanan siber dan berpartisipasi dalam komunitas-komunitas terkait untuk berbagi pengetahuan, pengalaman, dan belajar dari para ahli.

Sebagai Penulis Lepas

Selain bidang teknologi dan keamanan siber, saya juga aktif sebagai penulis lepas. Saya menulis artikel tentang berbagai topik, termasuk teknologi, pendidikan, dan kehidupan kampus. Menulis adalah salah satu cara saya untuk berbagi pengetahuan dan pandangan saya dengan orang lain, serta meningkatkan keterampilan komunikasi saya.

Proyek dan Portofolio

Di portofolio ini, Anda akan menemukan beberapa proyek yang telah saya kerjakan selama ini. Proyek-proyek ini mencerminkan perjalanan belajar saya dan dedikasi saya terhadap pengembangan web dan keamanan siber. Saya selalu terbuka untuk menerima masukan dan kritik yang membangun untuk terus memperbaiki diri.

Terima kasih telah mengunjungi portofolio saya. Jangan ragu untuk melihat-lihat proyek saya dan menghubungi saya jika Anda memiliki pertanyaan atau ingin bekerja sama. Saya sangat menantikan kesempatan untuk belajar dan tumbuh bersama komunitas pengembang dan keamanan siber yang luar biasa.
